# 테마 적용 빠른 가이드

## 방법 1: PaperMod 테마 적용 (추천)

### 설치
```bash
# Git Bash 또는 터미널에서
git submodule add https://github.com/adityatelange/hugo-PaperMod.git themes/PaperMod
```

### 설정
`hugo.toml` 파일 수정:
```toml
theme = "PaperMod"

[params]
  env = "production"
  title = "My New Hugo Site"
  description = "Hugo 사이트입니다"
  
[menu]
  [[menu.main]]
    identifier = "home"
    name = "홈"
    url = "/"
    weight = 10
  [[menu.main]]
    identifier = "about"
    name = "소개"
    url = "/about"
    weight = 20
  [[menu.main]]
    identifier = "posts"
    name = "블로그"
    url = "/posts"
    weight = 30
```

## 방법 2: Stack 테마 적용

### 설치
```bash
git submodule add https://github.com/CaiJimmy/hugo-theme-stack.git themes/stack
```

### 설정
```toml
theme = "stack"
```

## 방법 3: 현재 커스텀 디자인 유지

현재 디자인이 이미 개선되었습니다! 더 개선하고 싶다면:
- `static/css/style.css` 파일을 수정하세요
- 색상, 레이아웃, 애니메이션 등을 추가할 수 있습니다

## 테마 적용 후

1. Hugo 서버 재시작
2. 브라우저에서 확인
3. 테마별 README.md 참고하여 추가 설정
