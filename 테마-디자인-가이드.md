# Hugo 테마 적용 및 디자인 가이드

Hugo 사이트에 테마를 적용하거나 디자인을 개선하는 방법을 안내합니다.

## 방법 1: 기존 Hugo 테마 적용하기

### 1단계: 테마 선택

인기 있는 Hugo 테마들:
- **PaperMod** - 깔끔하고 현대적인 블로그 테마
- **Stack** - 미니멀한 카드 스타일 테마
- **Ananke** - 간단하고 빠른 테마
- **LoveIt** - 기능이 풍부한 테마
- **Hugo Book** - 문서 사이트용 테마

테마 갤러리: https://themes.gohugo.io/

### 2단계: 테마 설치 방법

#### 방법 A: Git Submodule 사용 (권장)

```bash
# themes 디렉토리 생성 (없는 경우)
mkdir themes

# 예: PaperMod 테마 설치
git submodule add https://github.com/adityatelange/hugo-PaperMod.git themes/PaperMod

# 또는 Stack 테마
git submodule add https://github.com/CaiJimmy/hugo-theme-stack.git themes/stack
```

#### 방법 B: 직접 다운로드

1. 테마의 GitHub 페이지에서 ZIP 파일 다운로드
2. 압축 해제 후 `themes/테마이름` 폴더에 복사

### 3단계: 테마 적용

`hugo.toml` 파일에서 테마 설정:

```toml
theme = "PaperMod"  # 또는 설치한 테마 이름
```

### 4단계: 테마별 설정

각 테마마다 고유한 설정이 있습니다. 테마의 `README.md` 파일을 참고하세요.

예시 (PaperMod):
```toml
[params]
  env = "production"
  title = "My New Hugo Site"
  description = "Hugo 사이트입니다"
  
[menu]
  [[menu.main]]
    identifier = "home"
    name = "홈"
    url = "/"
    weight = 10
```

## 방법 2: 커스텀 디자인 개선하기

현재 커스텀 디자인을 더 개선할 수 있습니다.

### 현재 디자인 개선 옵션

1. **색상 테마 추가**
2. **반응형 디자인 개선**
3. **애니메이션 효과 추가**
4. **다크 모드 지원**
5. **폰트 및 타이포그래피 개선**

## 빠른 시작: 간단한 테마 적용 예시

### PaperMod 테마 적용하기

```bash
# Git이 설치되어 있다면
git submodule add https://github.com/adityatelange/hugo-PaperMod.git themes/PaperMod
```

그 다음 `hugo.toml`에 추가:
```toml
theme = "PaperMod"
```

## 테마 vs 커스텀 디자인

| 구분 | 테마 사용 | 커스텀 디자인 |
|------|----------|--------------|
| **장점** | 빠른 시작, 전문적 디자인, 많은 기능 | 완전한 제어, 독특한 디자인 |
| **단점** | 커스터마이징 제한, 테마 의존성 | 개발 시간 필요, 모든 기능 직접 구현 |
| **추천** | 블로그, 빠른 프로토타입 | 브랜드 사이트, 특별한 요구사항 |
