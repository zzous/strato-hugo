# Hugo 실행 방법

Windows에서 Hugo 서버를 실행하는 여러 방법입니다.

## 방법 1: PowerShell 또는 CMD에서 직접 실행

1. **PowerShell** 또는 **CMD**를 관리자 권한으로 열기
2. 프로젝트 디렉토리로 이동:
   ```powershell
   cd C:\strato\project\kt-cloud-next-console
   ```
3. Hugo 실행:
   ```powershell
   & "C:\Users\$env:USERNAME\AppData\Local\Microsoft\WinGet\Packages\Hugo.Hugo.Extended_Microsoft.Winget.Source_8wekyb3d8bbwe\hugo.exe" server
   ```
   또는 (사용자 이름이 smile인 경우):
   ```powershell
   "C:\Users\smile\AppData\Local\Microsoft\WinGet\Packages\Hugo.Hugo.Extended_Microsoft.Winget.Source_8wekyb3d8bbwe\hugo.exe" server
   ```

## 방법 2: PATH 환경 변수에 추가 (영구적 해결)

1. **시스템 속성** → **환경 변수** 열기
2. **시스템 변수**에서 `Path` 선택 → **편집**
3. **새로 만들기** 클릭 후 다음 경로 추가:
   ```
   C:\Users\smile\AppData\Local\Microsoft\WinGet\Packages\Hugo.Hugo.Extended_Microsoft.Winget.Source_8wekyb3d8bbwe
   ```
   (사용자 이름이 다르면 `smile`을 본인의 사용자 이름으로 변경)
4. **확인** 클릭하여 저장
5. 터미널을 **재시작**한 후:
   ```bash
   hugo server
   ```

## 방법 3: Git Bash에서 실행

Git Bash에서 실행하려면:

```bash
# 방법 A: 따옴표 사용 (권장)
"/c/Users/smile/AppData/Local/Microsoft/WinGet/Packages/Hugo.Hugo.Extended_Microsoft.Winget.Source_8wekyb3d8bbwe/hugo.exe" server

# 방법 B: 전체 경로 사용
/c/Users/smile/AppData/Local/Microsoft/WinGet/Packages/Hugo.Hugo.Extended_Microsoft.Winget.Source_8wekyb3d8bbwe/hugo.exe server
```

## 방법 4: 배치 파일 생성 (가장 편리)

프로젝트 루트에 `run-hugo.bat` 파일을 만들고:
```batch
@echo off
cd /d "%~dp0"
"C:\Users\smile\AppData\Local\Microsoft\WinGet\Packages\Hugo.Hugo.Extended_Microsoft.Winget.Source_8wekyb3d8bbwe\hugo.exe" server
pause
```

이 파일을 더블클릭하면 서버가 실행됩니다.

## 서버 실행 후

서버가 성공적으로 실행되면:
- 브라우저에서 `http://localhost:1313` 접속
- 콘솔에서 `Ctrl+C`로 서버 중지

## 문제 해결

만약 "hugo: command not found" 오류가 발생하면:
1. Hugo가 실제로 설치되어 있는지 확인:
   ```powershell
   Test-Path "$env:LOCALAPPDATA\Microsoft\WinGet\Packages\Hugo.Hugo.Extended_Microsoft.Winget.Source_8wekyb3d8bbwe\hugo.exe"
   ```
2. 설치되어 있지 않다면:
   ```powershell
   winget install Hugo.Hugo.Extended
   ```
3. 터미널을 완전히 종료하고 다시 열기
